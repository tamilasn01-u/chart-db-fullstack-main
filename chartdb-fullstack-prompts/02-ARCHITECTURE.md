## ğŸ¯ Key Design Decisions

### 1. Optimistic UI Updates
- Update UI immediately on user action
- Sync to server in background
- Rollback on failure

### 2. Throttled Position Updates
- Throttle to 50ms during drag
- Persist final position on drag end
- Reduces server load by 95%

### 3. Event Sourcing for Collaboration
- All changes are events
- Events are broadcast to all clients
- Enables undo/redo and history

### 4. Hybrid Storage Strategy
- Primary: PostgreSQL (source of truth)
- Cache: Redis (presence, cursors)
- Fallback: IndexedDB (offline)

### 5. JWT with Refresh Tokens
- Short-lived access tokens (15 min)
- Long-lived refresh tokens (7 days)
- Secure WebSocket authentication
